# Modern Pause Menu

A simple, animated pause menu for FiveM.
It fixes the map bug and includes anti-spam protection.

**Discord Support:** (https://discord.gg/VUdFdARrWd)

**Folder Name:** # blossom_pause

## Installation
1. Put the folder into `resources`.
2. Add `ensure blossom_pause` to your `server.cfg`.
3. Edit `config.lua`.
